## キー操作

### Title
- Enterで次の画面へ

### Stage Select
- A,Dで左右選択
- Enterで決定，次の画面へ

### Game
- A,Dまたは左右矢印で左右移動
- Spaceでジャンプ
- NUM1,2,3でそれぞれに応じた拡縮
