### 操作

|動き|キーボード操作|Xboxコントローラ|
|---|---|---|
|左移動|A|十字左キー|
|右移動|D|十字右キー|
|ジャンプ|SPACE|Aボタン|
|縮小|J|RBボタン|
|通常|K|Xボタン|
|拡大|L|LBボタン|
|アイテムを持つ/離す|I|Bボタン|
